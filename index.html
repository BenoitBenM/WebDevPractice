Nihil: My Webdev Projects

  2. French flag

    • In this part I practiced with positioning and creating shapes

  3. Make the cat slimmer!

    • This was a great and fun way to learn about media querries and responsive websites.

    • You have to make the page narrower to help the cat get slimmer.

  4. Pricing table

    • I had to use flexbox in order to make dynamic pricing option boxes that automatically adapt to the window size
